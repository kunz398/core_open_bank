# core_open_bank
this project is aimed to provide a core banking which is completly open source with all features ranging from, tellers, treasury , loans, atm connectivity, and other payment gateway

Stack
+-------------------+----------------------------------------------+
| Layer             | Technology                                   |
+-------------------+----------------------------------------------+
| Framework         | Spring Boot 4.0.3                            |
| Language          | Java 21                                      |
| Database          | PostgreSQL + Spring Data JPA (Hibernate)     |
| Security          | Spring Security + JWT (jjwt 0.12.6)          |
| Password Hashing  | Argon2 via Bouncy Castle                     |
| Validation        | Jakarta Validation                           |
| Boilerplate       | Lombok                                       |
| Config            | .env via spring-dotenv                       |
+-------------------+----------------------------------------------+
